## Usage

1. Create `.env` file:

```
IG_LOGIN=email
IG_PASSWORD=password
```

2. Create a symlink to your photos folder:

```
ln -s /my/photos/folder photos
```

3. Add already published photos to `data/published.json`.

4. Install dependencies:

```
npm install
```

5. Run:

```
npm start
```
